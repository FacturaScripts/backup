# Backup
Plugin de copias de seguridad para FacturaScripts.
- https://facturascripts.com/plugins/backup

Permite crear y restaurar copias de seguridad de la base de datos (MySQL y PostgreSQL) y de los archivos de FacturaScripts. Añade la sección **Copia de seguridad** al menú Administrador y ofrece:

- Copia de la base de datos y de los archivos, con descarga directa.
- Restauración de la base de datos y de los archivos desde una copia.
- Copias de seguridad **automáticas** programadas (frecuencia diaria, semanal o mensual, con día y hora configurables) a través del cron.
- Límite del número máximo de copias que se conservan, borrando automáticamente las más antiguas.
- Aviso cuando han pasado más de 30 días desde la última copia de seguridad.

## Carpeta
La carpeta debe llamarse igual que el plugin: **Backup**.
